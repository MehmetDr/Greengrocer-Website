# 🥦 Greengrocer Website

Modern, kullanıcı dostu ve dinamik bir **manav (greengrocer) web sitesi** projesidir.  
Bu proje, kullanıcıların taze meyve ve sebzeleri kolayca inceleyip satın alabileceği bir e-ticaret deneyimi sunmayı amaçlar.

---

## 🚀 Demo

🔗 Canlı Site:  
https://mehmetdr.github.io/Greengrocer-Website/

---

## 📌 Proje Amacı

Bu proje ile:

- Küçük ölçekli manavlar için dijital çözüm üretmek  
- Kullanıcı dostu bir alışveriş deneyimi sunmak  
- Modern web teknolojileri ile responsive tasarım geliştirmek  
- Frontend geliştirme becerilerini güçlendirmek  

amaçlanmıştır.

---

## 🧩 Özellikler

- 🛒 Ürün listeleme (meyve & sebzeler)
- 🔍 Ürün detay görüntüleme
- 📱 Responsive (mobil uyumlu) tasarım
- 🎨 Modern ve sade UI/UX
- ⚡ Hızlı ve optimize edilmiş yapı
- 🖼️ Görsellerle desteklenmiş ürün kartları
